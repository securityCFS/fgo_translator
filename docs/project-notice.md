# FGO Translator 项目说明

FGO Translator 是一个面向《Fate/Grand Order》剧情阅读与翻译的网页工具。它的目标是让玩家更方便地检索关卡脚本、查看不同服务器的已有译文，并在需要时使用自己的翻译接口生成临时翻译结果。

## 数据来源与翻译能力

本程序基于 Atlas Academy 的开放数据库构建。关卡、任务、脚本、角色立绘、背景与演出信息等检索数据均来自公开可访问的数据源。

网页会读取 Atlas 数据中的游戏脚本，并通过用户配置的 VLM / LLM API 接口提供实时翻译体验。对于国服、台服、北美服、韩服等已经同步或公开的官方翻译结果，程序会优先提供可选的 Atlas / 已有翻译入口，让用户可以直接加载对应服务器已有译文，而不必重复调用自己的 API。

## 隐私与凭据

用户配置的 auth token、API key、base URL 和模型名称仅保存在浏览器本地存储中。项目不会把这些凭据提交到仓库，也不会把凭据同步到 GitHub。

翻译后的脚本内容可能会被同步到本项目使用的 GitHub 托管仓库，用于缓存、共享和减少重复翻译请求。请不要在翻译内容中加入任何私人信息、密钥或不希望公开的文本。

## 素材与功能边界

页面展示的美术素材、图标、背景、角色图像等均来自公开数据库中可访问的资源页面。本项目只读取和展示公开资源，并不提供任何游戏解包、客户端修改、资源提取、账号相关操作或绕过游戏机制的功能。

游戏模式仅用于把公开脚本与翻译结果以视觉小说形式播放，方便阅读和校对，不会修改游戏本体或与游戏客户端交互。

## 支持原作

本项目仅作为玩家学习、阅读与翻译辅助工具。希望各位继续支持《Fate/Grand Order》及其官方内容。
